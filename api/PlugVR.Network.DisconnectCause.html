Enum DisconnectCause

Namespace: PlugVR.Network
Assembly: cs.temp.dll.dll
Syntax
public enum DisconnectCause

Fields

Name Description
ApplicationQuit

응용 프로그램이 종료되는 경우에 사용됩니다. 새 장면을 로드하지 않거나 OnDisconnected에서 다시 연결하지 않는 데 유용할 수 있습니다.
AuthenticationTicketExpired

인증 티켓은 다른 인증 서비스 호출 없이 모든 Photon Cloud 서버에 대한 액세스를 제공해야 합니다. 그러나 티켓이 만료되었습니다.
ClientTimeout

OnStatusChanged: 이 클라이언트는 서버의 응답이 적시에 수신되지 않았음을 감지했습니다.
CustomAuthenticationFailed

OnOperationResponse: 유효하지 않은 클라이언트 값 또는 클라우드 대시보드의 사용자 정의 인증 설정으로 Photon 클라우드에서 인증합니다.
DisconnectByClientLogic

OnStatusChanged: 클라이언트가 논리(C# 코드) 내에서 연결 해제되었습니다.
DisconnectByDisconnectMessage

클라이언트가 서버에서 "연결 해제 메시지"를 받았습니다. 자세한 내용은 디버그 로그를 확인하십시오.
DisconnectByOperationLimit

클라이언트가 작업을 너무 자주 호출하고 OperationLimit에 도달하여 연결이 끊어졌습니다. 이렇게 하면 클라이언트 측 연결 해제도 트리거됩니다.
DisconnectByServerLogic

OnStatusChanged: 서버가 방의 논리(C# 코드) 내에서 이 클라이언트의 연결을 끊었습니다.
DisconnectByServerReasonUnknown

OnStatusChanged: 서버가 알 수 없는 이유로 이 클라이언트의 연결을 끊었습니다.
DnsExceptionOnConnect

OnStatusChanged: 호스트 이름에 대한 Dns 확인에 실패했습니다. 이에 대한 예외는 오류 수준으로 포착되고 기록됩니다.
Exception

OnStatusChanged: 일부 내부 예외로 인해 소켓 코드가 실패했습니다. 로컬로 연결을 시도했지만 서버를 사용할 수 없는 경우 이런 일이 발생할 수 있습니다. 의심스러운 경우: Exit Games에 문의하십시오.
ExceptionOnConnect

OnStatusChanged: 서버를 사용할 수 없거나 주소가 잘못되었습니다. 포트가 제공되고 서버가 작동 중인지 확인하십시오.
InvalidAuthentication

OnOperationResponse: 유효하지 않은 AppId로 Photon 클라우드에서 인증합니다. 구독을 업데이트하거나 Exit Games에 문의하세요.
InvalidRegion

OnOperationResponse: 앱의 Photon Cloud 구독이 일부(다른) 지역에 잠겨 있을 때 인증합니다. 구독 또는 마스터 서버 주소를 업데이트하십시오.
MaxCcuReached

OnOperationResponse: CCU 버스트 없이 Photon 클라우드 구독을 사용할 때 인증(일시적으로) 실패했습니다. 구독을 업데이트하세요.
None

오류가 추적되지 않았습니다.
OperationNotAllowedInCurrentState

OnOperationResponse: 이 클라이언트에 대해 (현재) 사용할 수 없는 작업입니다(일반적으로 승인되지 않음). op 인증에 대해서만 추적됩니다.
ServerAddressInvalid

OnStatusChanged: 서버 주소가 불법적으로 IPv4로 구문 분석되었습니다. 불법 주소는 다음과 같습니다. 192.168.1.300. IPAddress.TryParse()는 이 작업을 통과시키지만 검사는 통과하지 않습니다.
ServerTimeout

OnStatusChanged: 시간 초과로 인해 서버에서 이 클라이언트의 연결을 끊었습니다(클라이언트의 승인 누락).
